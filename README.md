# h-player-v2 mobile版

## Get Started

### android
android sdk,android studio

```bash
# 安装依赖
quasar mode add capacitor
# 启动开发服务器
quasar dev -m capacitor -T android
# 编译二进制包
quasar build -m capacitor -T android
```
